void TestLoadBalancer::simulateLoadBalancer ( int  initRequestAmount,
int  initServerAmount,
int  clockCycles 
)
static

Simulate load balancing with an initial amount of requests, initial server amount, and clock cycles this load balancer will be running for.

As the simulation runs, there will be new requests added to the request queue every 10 to 150 clock cycles.

Parameters
initRequestAmountInitial amount of requests the request queue will start out with
initServerAmountInitial amount of servers the server group will start out with
clockCyclesThe amount of clock cycles the simulation will run for
