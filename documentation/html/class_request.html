◆ Request() [1/2]

Request::Request ( )

Construct a new Request object with random input & output IP addresses and random duration the request would take to process.

IP addresses use the IPv4 format, meaning the it uses the "x.x.x.x" format with each x being a value from 0 to 255. The duration of each request ranges from 10 to 300 clock cycles.
